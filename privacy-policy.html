Privacy Policy

Effective Date: 16 February 2026
Application Name: Monify

1. Overview

Monify (“we”, “our”, or “the application”) is committed to safeguarding user privacy. This Privacy Policy explains how information is handled when users access and use the application.

2. Information We Do Not Collect

Monify does not collect, transmit, process, or store personal data on external servers. The application does not require account creation and does not request:

  • Bank account information
  • Credit or debit card details
  • Government-issued identification
  • Login credentials
  • Personal financial records outside user input

3. Local Data Storage

All expense data entered by users is stored locally on the user’s Android device using secure local database storage mechanisms. No data is uploaded to cloud services or shared with third parties.

4. Data Usage

Data entered within the application is used exclusively to:

  • Generate expense summaries
  • Create weekly and monthly reports
  • Provide in-app financial insights
  • Enable reminder notifications

5. Permissions

Monify may request limited system permissions necessary for core functionality:

  • Notification Permission – Used solely for reminder alerts.

No background tracking or unauthorized data access occurs.

6. Third-Party Services

Monify does not integrate:

  • Advertising SDKs
  • Analytics tracking services
  • Behavioral profiling tools
  • External data monetization platforms

7. Data Security

Since all data remains on the user’s device, data security depends on the device’s built-in protections. Users are encouraged to enable device-level security measures such as screen lock or biometric authentication.

8. User Rights

Users maintain full control over their data. At any time, users may:

  • Delete application data
  • Uninstall the application
  • Clear stored data via device settings

9. Children’s Privacy

Monify does not knowingly collect information from children under the age of 13. The application is intended for general audiences.

We reserve the right to modify this Privacy Policy to reflect updates in functionality or regulatory requirements. Any changes will be updated on this page with a revised effective date.
